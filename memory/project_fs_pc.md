Приложение оценки проектов (FS_PC) — React + Node.js + SQLite, Вариант A (локальный сервер).

**Why:** Для команды 5-7 оценщиков в одной компании, с историей изменений по полям.

**Структура:**
- `C:\YandexDisk\Обучение\Вайб_кодинг\FS_PC\`
- `server/` — Node.js + Express + better-sqlite3, порт 3001
- `client/` — React + Vite + TypeScript + Tailwind CSS, порт 5173
- `data/projects.db` — SQLite база
- `start.ps1` — запуск обоих серверов

**Запуск:** `.\start.ps1` из корня проекта

**Формулы расчёта:**
- трудозатраты = дни × 8 × Σ(загрузки ролей) / 100
- фонд = трудозатраты × ставкаЧасаРуб
- резерв = фонд × 10%
- УСН = фонд + резерв + компенсацияПродаж
- КП = УСН × 1.07 (НДС)
- С рисками = УСН × (1 + рискЭтапа)

**Справочники рамок/результатов/документов:**
- Таблица `work_references` — author_id ссылается на users
- NULL author = "Базовый" (из каталога)
- Фильтр по автору в модале ReferenceSelector

**How to apply:** При доработках помнить что расчёт делается и на клиенте (utils/calc.ts) для мгновенного отклика, и на сервере (src/calc.ts) для сохранения.
